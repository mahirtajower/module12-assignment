# Module 12 Assignment: Business Analytics Fundamentals and Applications

This repository contains my work for Module 12. The assignment focuses on applying business analytics techniques to sales, customer, store, and operational data for a fictional grocery company called GreenGrocer.

## Files Included

- `module12-assignment.py`

## What I Practiced

In this assignment, I practiced business analytics using Pandas, NumPy, Matplotlib, and SciPy. I analyzed sales performance, customer segments, store efficiency, seasonal patterns, correlations, regression predictions, department trends, and profit opportunities. I also practiced creating visualizations and using data to support business decision-making.
